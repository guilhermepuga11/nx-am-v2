How it works. A rule fires on its day-of-month (Europe/Lisbon) from a daily check — Apps Script has no native "Nth of the month", so the day is set to 1–28. Each partner gets the template in their own language (English fallback optional). A partner already sent this month is never sent again, so a re-run or a retry is safe. If the template, segment, transport, or recipient cap isn't right, the rule sends nothing and records why. Always Preview before relying on a new rule.
